# javascript-pong

JavaScript Pong
Created by: Chris Berger
Date: July 27, 2020

Features:

* Computer Enemy, trained to follow ball
  * Difficulty level can be changed by modifying enemy speed. (Higher = More Difficult)
* Player, enemy and opponent classes
* Click to Restart
